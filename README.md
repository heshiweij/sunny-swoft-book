## 部署

Clone 项目

```$xslt
git clone ...
```

NPM 安装 Gitbook

```$xslt
npm install gitbook-cli -g
```

启动

```$xslt
gitbook serve
```

更多用法

[gitbook-cli](https://github.com/GitbookIO/gitbook-cli)
